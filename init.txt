#GIT INIT => Inicialmente tenemos un directorio de trabajo, dentro de esta carpeta ejecutamos el comando git init => eso crea dos cosas el staging y repositorio local
git init => Crea un area en memoria RAM que se llama staging, es un area completamente desconectado, donde inicialmente se van agregando los cambios.
         => Tambien se crea el repositorio, es la carpeta donde se encuentra el archivo /.git/ aqui se encuentran todos los cambios al final del proyecto.
